Выбор строки

Каждая строка DataFrame соответствует точке данных в пространстве задачи. Чтобы создать подмножество из
имеющихся элементов данных, необходимо выбрать строки. Существуют два метода создания подмножества:
- указать расположение строк;
- задать критерии фильтра

Подмножество строк может быть получено по расположению следующим образом:

>>> df.iloc[1:3,:]
 id - name - age - desicion
1 - 2 - Elena - 23 - False
2 - 3 - Steven - 40 - True

Данный код вернет первые две строки и все столбцы.

Чтобы создать подмножество с помощью фильтра, мы должны указать критерии выбора в одном или нескольких
столбцах. Это происходит следующим образом:

>>> df[df.age>30]
 id - name - age - decision
0 - 1 - Fares - 32 - True
2 - 3 - Steven - 40 - True

>>> df[(df.age<35)&(df.decision==True)]
 id - name - age - decision
0 - 1 - Fares - 32 - True

Обратите внимание, что этот код создает подмножество строк, удослетворяющее условию, указанному в фильтре.
