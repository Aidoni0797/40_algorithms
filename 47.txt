Обмен значений переменных в Python

При реализации алгоритмов сортировки и поиска мы сталкиваемся с необходимостью обмена значений двух
переменных между собой. В Python для этого есть простой способ:

var1 = 1
var2 = 2
var1, var2 = var2, var1
>>> print(var1, var2)
>>> 2 1

Давайте посмотрим, как это работает

Данный способ обмена значений используется во всех алгоритмах сортировки и поиска, о которых мы поговорим
в этой главе.
